# ExpenseX

A Personal Finance Dashboard built using HTML, CSS and Vanilla JavaScript.

## Features

- Dashboard
- Analytics
- Transactions
- Budgets
- Goals
- Reports
- Calendar
- Charts
- Dark Mode
- LocalStorage
- Income & Budget Management
- Expense Tracking (CRUD)
- Dashboard Analytics
- Smart Financial Insights
- Financial Health Score
- Monthly Budget Comparison
- Top 5 Largest Expenses
- Category-wise Expense Charts
- CSV Export
- Search & Filter Transactions
- Responsive Dark UI

## Tech Stack

- HTML5
- CSS3
- JavaScript
